What is ChainsMakie.jl? â€‹

MCMC chains are common objects in probabilistic programming languages such as Turing.jl. Inspecting the MCMC chains visually is a key part of the workflow, and ChainsMakie.jl provides a set of tools to simplify this task.

julia
using ChainsMakie, CairoMakie
import MCMCChains: Chains

chains = Chains(randn(300, 3, 3), [:A, :B, :C])

plot(chains)
